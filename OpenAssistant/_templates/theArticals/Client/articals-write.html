How to make Project and App in Django

This is django's very basic artical, who instruct us about django implementations initial steps.

I explain you a situation from scratch, I make project named OpenAssistant , So first i give you instructions about what is the must steps after make app. And then give you an idea about after that how we make an app named Admin, Client, Home, API. How to make Django Projects

See first step to you have such requirements, to make Django Project on your system. list in below .....

  • Python

  • Django Rest Framework

Project making process start from the directory/folder location, where we want to make project. Suppose location is F:\django\projects.
So, now there are two ways .....

  • First thing that we go to the Command Prompt (cmd), and there is you write below instructions to change your directory location.
